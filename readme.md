# travel-app-capstone
Is a travel app that gets the location weather and picture of the destination. 

## Dependencies

```bash
npm install 
```
## How to run the project
To run server 
```bash
Npm run start
```
Development 
```bash
npm run build-dev
```
Production 
```bash
npm run  build-prod
```
